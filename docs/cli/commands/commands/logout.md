# inigo login
---

## Commands
- [inigo logout](#inigo-logout)

---

### ```inigo logout```
#### **Description**
Deauthenticates the session from Inigo.

#### **Syntax**
```
inigo logout
```

#### Optional flags
* `--help`, `-h`
show help info


#### **Output**
```
$ inigo logout
logged out
```
